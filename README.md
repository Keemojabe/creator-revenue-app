# Creator Revenue Calculator

**Creator Revenue Calculator** is a fully functional web app that helps online creators **project their revenue from ads, affiliates, and brand deals**.  
It includes advanced features like **growth projections, chart visualization, animated counters, PDF exports, and a Pro upgrade modal**.

---

## 🚀 Features

### Free Version
- Input monthly views, CPM, affiliate CTR, product price, and brand deals
- Instant revenue calculation
- Animated number count-up effect

### Pro Version
- 12-month revenue growth projections
- Line chart visualization (powered by Chart.js)
- Download PDF report (Pro only)
- Modal-based paywall with Lemon Squeezy integration for seamless upgrades
- Animated UI with hover effects and modern dashboard design

### Landing Page
- Clean landing page to promote your app
- Links to Free and Pro versions
- Fully responsive and professional
